## 修复 Waline 不随浅深色模式切换（2026-03-29）

**根因**：`dark` 参数传的是布尔值快照（`theme === "dark"`），初始化后不再响应主题变化。

**修复**：改为传 CSS 选择器字符串 `dark: 'html[data-theme="dark"]'`，Waline 内部自动用 MutationObserver 监听 `<html>` 属性变化，主题切换时自动同步，无需任何额外代码。同时删除了无用的 `getSiteTheme()` 函数。
